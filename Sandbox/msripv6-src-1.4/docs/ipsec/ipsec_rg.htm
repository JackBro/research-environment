Range Selector Example

A range value is used to cover certain traffic. The previous example, used specific values for the RemoteIPAddr. Here, the selector in Zealand's SPD is + 3ffe:1a01:4:b::-3ffe:1a01:4:b:ffff::. Any traffic within the global prefix "3ffe:1a01:4:b::", is afforded IPSec protection. Ranges in this implementation must be entered from the start of range to the end of range, where start is the smaller of the values. Ranges can be used on all the selectors except the Protocol selector.
